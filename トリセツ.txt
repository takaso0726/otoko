【フォルダ】
🔴 【最終確認】これだけは絶対に選択から外してください！
以下の3つのフォルダと、その中身は消すとプロジェクトが壊れる（データが消滅する）ので、絶対に削除しないよう注意してください。

🔴 Assets / （プログラム、3Dモデル、画像、音、ステージデータなど全て）

🔴 ProjectSettings / （ゲームの設定、ボタン割り当てなど）

🔴 Packages / （導入しているプラグインの管理データ）
下消していいよ

🟢 Library / （容量削減に最も効果的。数GB〜数十GB空くことがあります）

🟢Temp / （一時ファイル）

🟢Logs / （ログデータ）

🟢 UserSettings / （ウィンドウ配置などのローカル設定）

🟢obj / （コンパイル時の一時フォルダ）

🟢 .vs / （Visual Studioの隠しフォルダ。環境によっては非表示になっています）

🟢 Build / または Builds / （過去に出力したゲームの実行ファイル）

【ファイル】

🟢*.sln （拡張子が .sln のソリューションファイルすべて）

🟢 *.csproj （Assembly-CSharp.csproj など、拡張子が .csproj のファイルすべて）
